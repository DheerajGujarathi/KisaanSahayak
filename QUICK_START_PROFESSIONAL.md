## 🚀 Full System Start

To run the complete application:

```powershell
# Terminal 1 - AI Engine
cd D:\RAG-Farmers\ai-engine
python app.py

# Terminal 2 - Backend
cd D:\RAG-Farmers\backend
npm start

# Terminal 3 - Frontend
cd D:\RAG-Farmers\frontend
npm start
```

Or use the startup script:
```powershell
.\start-all.bat
```
